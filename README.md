# YouTube_Clone
# 🎥 YouTube Clone  A modern YouTube Clone built with **React.js**, **Vite**, **CSS Modules**, and the **YouTube Data API v3**. 
# This project replicates the core functionality and user interface of YouTube, providing a responsive and interactive video streaming experience.  
## ✨ Features  
* 🏠 YouTube-style Home Page
* 📺 Trending videos fetched using the YouTube Data API
* 🔍 Search bar UI
* 📂 Collapsible sidebar (Home, Shorts, Subscriptions, History)
* ▶️ Video Player page with embedded YouTube videos
* 📊 Real-time video details (title, views, likes, description)
* 📱 Responsive and clean user interface
* 🎨 Modern dark theme inspired by YouTube
* ⚡ Fast development using Vite

## 🛠️ Tech Stack  
* React.js
* Vite
* CSS Modules
* React Router DOM
* Lucide React Icons
* YouTube Data API v3

## 🚀 Getting Started  
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Add your YouTube Data API key to a `.env` file:     ```    VITE_YOUTUBE_API_KEY=YOUR_API_KEY    ```
4. Start the development server:     ```    npm run dev    ```
